# 2026-06-29 — Wallet Solution overview, scaffolded

## What this is
A new single-file case study (`index.html`) for the **BillDesk Wallet Solution**
— a stored-value wallet shipped as an embeddable SDK + an admin portal, themed
for Mumbai Metro and FedEx. Built with the `project-overview` skill, reusing the
BillDesk design system (`<style>` block) from the Netbanking 2.0 overview so the
two docs are visually one family.

## Source material
15 dropped PNGs in `Screens/`. Viewed each, then organised by feature into
`assets/sdk/` (6), `assets/whitelabel/` (4), `assets/admin/` (4) with web-clean
kebab names. `Enter amount (1).png` was a dup of `Enter amount.png` — used one.

## The story
Linear 10-section arc: hero → overview → problem → goals → **Unit 01 Wallet SDK**
(wallet screen low/funded, top-up pipeline + sheet→amount, one-tap pay → metro QR
ticket, 3 design decisions, mobile+desktop) → **Unit 02 Whitelabel/FedEx** (same
SDK re-skinned, quote-aware top-up + 2× upsell, branded sign-up) → **Unit 03
Admin** (dashboard, merchants, spend ledger, remittance + recon guide) → scaling
→ impact → future → reflection. Spine: *fund once → pay instantly → operator
settles.* Impact kept honest — two live whitelabels, mobile+desktop, end-to-end
admin; no invented numbers.

## Build mechanics
- Assembled from three temp parts: head (title + Google Fonts) + extracted
  749-line `<style>` from the Netbanking doc + hand-written body. `cat`'d into
  `index.html` (~1320 lines).
- Components reused: hero-grid, section-head/sec-num, cards-grid, clean lists,
  unit-divider/unit-kicker, ba-grid phone two-ups, figure/frame/placeholder,
  pipeline, decision blocks, step-callout, lightbox.
- Verified: all 14 asset paths resolve; single `<style>` tag; inline lightbox
  script JSC exit=0 (Esc-to-close added).

## Hero tweak (per Mridhul)
"Keep the image small, put it in a mockup." Replaced the full-bleed hero img with
a **CSS phone mockup** (`.phone-mock` — dark bezel, notch, rounded screen,
`object-fit:cover` top-anchored), 248px wide, soft orange radial glow behind it,
click-to-zoom. Stacks under 900px.
